# RLS

## RLSとNightly

　普通にnightlyをインストールするとrlsが使えない. これはVSCodeとかで使うので必要だったりする.


```bash
rustup install nightly
```

　実は対応表みたいなのがあってrlsに対応しているnightlyを導入する必要がある.[<sup>1</sup>](#com-1)
