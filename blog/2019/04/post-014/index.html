Hello World

　RustでHello Worldしてみましょう. 関数名は基本的にスネーク・ケース(snake case)です. 文字列はダブル・クオートで囲みます.1

fn say_greeting() {
  println("Hello World");
}

注釈

  1. 実はダブル・クオートで囲むだけでは文字列型ではなく&`static strという型らしいです.
