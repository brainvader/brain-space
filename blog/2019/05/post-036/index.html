2019-05-14

複数のバイナリを管理する

binファイルの作成

　binというファイルがいるようです.1

Cargo.tomlに複数のbinセクションを指定する.

　次にCargo.tomlに以下のように書く.

[[bin]]
name = "project_name"
src  = "src/main.rs"

[[bin]]
name = "second"
src  = "src/bin/second.rs"

　後はnameで指定した名前をオプションとして渡す.

cargo run --bin second
